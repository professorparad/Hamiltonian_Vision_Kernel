# Reconstruction Helpers

This folder contains utilities for turning predicted patches back into images.

## Files

- `patch_stitching.py`: places reconstructed patches back into image space.
- `seam_bleading.py`: blends seams between reconstructed patches.

The misspelling in `seam_bleading.py` is historical. Renaming it would require
updating imports, so leave it alone unless doing a deliberate cleanup pass.
